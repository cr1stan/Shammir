Manual de uso: Requisitos.
Al python 3.12 o almenos 3.0

a) Bilioteca Sympy
	Ejecutar directamente en la terminal del proyecto de Visual Studio el comando que comando pip:
	-pip install sympy
b) Biblioteca Cryptography 
	Ejecutar desde la terminal del proyecto de Visual Studio el comando pip:
	-pip install cryptography

Instrucciones para ejecutar los comando codificar y decodificar:

1)Codificar (-c): Ejecutar desde la terminal/bash en la ruta del proyecto python "nombre_del_proyecto.py" -c --archivo 'ruta_del_archivo.txt' --contraseña "nombre_de_la_constraseña"
Nota: Asegurar de no colocar la ruta completa si el archivo si se encuentra en la ruta local del proyecto

2)Decodificar (-d): Ejecutar desde la terminal/bash en la ruta del proyecto de python python bow.py -d --archivo_codificado 'archivo_codificado.aes' --fragmentos 'n_evaluaciones.frags'  


Nota general: El archivo codificado y decoficado se enontraran el la carpeta local donde se encuentre el archivo.txt
